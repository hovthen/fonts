# Fonts CDN

免费商用字体引用库

## LanaPixel

```css
@font-face {
 font-family: 'LanaPixel';
 src: url('https://cdn.jsdelivr.net/gh/hovthen/fonts@master/LanaPixel/LanaPixel.ttf')
  format('truetype');
}
```

来源：<https://opengameart.org/content/lanapixel-localization-friendly-pixel-font>
